In-Depth Python File Handling for Django Developers
1. Opening Files (open())
open() is used to access a file. It returns a file object for reading or writing.
Syntax:
file = open('filename.txt', 'mode')
Modes:
'r' - Read, 'w' - Write, 'a' - Append, 'x' - Create, 'b' - Binary, 't' - Text
2. Reading Files
with open('data.txt', 'r') as f:
    content = f.read()
with open('data.txt', 'r') as f:
    line = f.readline()
with open('data.txt', 'r') as f:
    lines = f.readlines()
with open('data.txt', 'r') as f:
    for line in f:
        print(line.strip())
3. Writing and Appending
with open('log.txt', 'w') as f:
    f.write("Overwriting the file.\n")
with open('log.txt', 'a') as f:
    f.write("Appending new log.\n")

7. CSV File Handling
import csv
with open('people.csv', 'w', newline='') as f:
    writer = csv.writer(f)
    writer.writerow(['Name', 'Age'])
    writer.writerow(['Ram', 25])
with open('people.csv', 'r') as f:
    reader = csv.reader(f)
    for row in reader:
        print(row)
